## Getting started

### Clone the repo
git clone https://github.com/jackiiechangg/Portfolio.git
cd Portfolio

### Plain HTML/CSS/JS
If your site is static:
- Open `index.html` in your browser
- To run a local static server:  
  python3 -m http.server 8000
- Visit http://localhost:8000

### React (Create React App)
Install dependencies and run:

npm install
npm start

Visit http://localhost:3000

### Next.js
Install dependencies and run:

npm install
npm run dev

Visit http://localhost:3000

## Build for production
- React (CRA): npm run build
- Next.js: npm run build && npm start

## Deployment
Common options:
- GitHub Pages: use `gh-pages` or GitHub Pages from the `main` branch (or `gh-pages` branch)
- Vercel: connect the repo in the Vercel dashboard and deploy automatically on push
- Netlify: connect and deploy automatically

## Customizing content
- Update your name, bio, and links in the header or `src`/`content` files (wherever your site stores them).
- Replace project entries in `/projects` (or the equivalent directory) with your projects, descriptions, and screenshots.
- Update contact info and links (email, LinkedIn, Twitter, resume).
